单纯形算法

单纯形法是通过设置不同的基变量，经过矩阵的线性变换，求得基的可行解（此时这个可行解在可行域的顶点处），并判断该解是否达到最优？否则继续设置下一组基向量，重复执行，直至找到最优解。本质上来说单纯形算法就是一个循环迭代的过程

单纯形算法是通过线性规划的标准型来求解的，要求：
1) 目标函数要求max
2) 约束条件均为等式
3) 决策变量为非负约束

而普通的线性规划如果要变成标准型
1) 对偶
2) 约束为不等式，则可以在左端加入非松弛变量，转化为等式

单纯形算法的几何意义

在标准型中，如果有m个约束条件（不包括非负约束），n个决策变量，且(n >= m)。首先我们选取m个基变量xj’ ( j = 1,2,…,m ), 基变量对应约束矩阵的列向量线性无关，通过矩阵的线性变换，基变量可以由非基变量表示，这时候再令非基变量等于0，可以求得基变量的值.

这样说可能太过抽象，我们来举个例子吧.

初始随便先选择x2, x3为基变量，令x1, x4等于0可以解基变量x2, x3的值.(这里的C代表的是目标函数)

这个时候令x1, x4的值为0，很容易就可以解出基变量x2 = 9/2 , x3 = 15/2
从几何意义上来说，x1 = 0 表示可行解在y轴上， x4 = 0 表示可行解在x1 + 2*x2 = 9的直线上，那么解除的点就代表着这两条直线的交点，也是可行域的顶点，如图：

这时候我们可以通过选择不同的基变量来获得不同的可行域顶点

如何判断哪个顶点是最优解

就像前面说的那样，我们用非基变量表示基变量

目标函数z也完全用非基变量来表示：

当达到最优解的时候，所有的σj应该小于等于0。当存在j使得 σj>0 时，当前解非最优。
因为当前的目标函数值为z0， 其中所有的非基变量值均为0， xj’=0 代表可行域的某个边界。 是x’j的最小值， 如果代表可行解的点逐渐离开这个边界， xj’会变大， 如果σj >0 显然目标函数z0的值也会变大，所以当前解不是最优解。
